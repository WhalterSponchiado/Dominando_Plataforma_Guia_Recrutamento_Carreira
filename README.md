# Dominando_Plataforma_Guia_Recrutamento_Carreira
Guia estratégico sobre a plataforma Gupy desenvolvido com Google NotebookLM, explorando metodologias de recrutamento, TRI e otimização de carreira através de Inteligência Artificial.


🚀 Guia de Recrutamento e Carreira com NotebookLM


📌 Sobre o Projeto
Este repositório é o resultado de um estudo intensivo sobre o ecossistema da Plataforma Gupy. Utilizando o Google NotebookLM como meu copiloto de aprendizado, consolidei informações dispersas em um "Caderno Temático" inteligente para desvendar como a IA de recrutamento avalia candidatos.
O objetivo é transformar a "caixa preta" da Gupy em estratégias claras para profissionais que buscam melhores colocações no mercado.


🧠 Insights Gerados pela IA
Através da curadoria de fontes (artigos, vídeos e documentos técnicos), o NotebookLM sintetizou três pilares fundamentais que você encontrará detalhados neste guia:
Extração de Dados (Parsing): Como a IA lê seu currículo. Prioridade para campos preenchidos na plataforma vs. o arquivo PDF.
Metodologia TRI (Teoria de Resposta ao Item): O segredo por trás dos testes de lógica. Por que "chutar" pode destruir sua pontuação.
Contexto Semântico: A Gupy entende sinônimos. "Gerente de Vendas" e "Sales Manager" são lidos como a mesma competência.


🛠️ Metodologia e Ferramentas
Google NotebookLM: Utilizado para centralizar fontes e gerar resumos estruturados.
Curadoria de Conteúdo: Foram processadas mais de 30 fontes (mesmo lidando com desafios de links expirados e erros 404, conforme documentado no histórico de fontes).
Prompt Engineering: Aplicação de perguntas específicas para extrair a lógica de "Ranking" e "Fit Cultural".


📁 Estrutura do Repositório
/documentos: Resumos gerados pelo NotebookLM em formato Markdown.
/prompts: Lista de comandos utilizados para "treinar" o caderno sobre o tema.
Guia_Gupy_Resumo.pdf: O guia consolidado para leitura rápida.


💡 Como utilizar este Guia
Leia o arquivo principal sobre Parsing de Dados para otimizar seu currículo hoje.
Consulte a seção de Testes e Mapeamento antes de iniciar qualquer processo seletivo na plataforma.
Utilize os prompts de revisão para checar se seu currículo está "legível" para uma ATS (Applicant Tracking System).
